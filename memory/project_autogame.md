## Core Identity
- **Project Name:** AutoGame / GameGPT / Magiscapes (麦琪的花园)
- **Vision:** "AI-defined game development paradigm", "Full-element generation", "Infinite Game".
- **Slogan:** "Let Every Inspiration Become a Game" (让每一个灵感都变成游戏).

## Key Technologies
### 1. GameGPT Framework
- **Definition:** World's first AI game development agent framework.
- **Metrics:** Reduces dev cycle by 95%, cost by 90%, manpower by 75%.
- **Architecture:** Multi-agent collaboration (Manager, Engineer, Reviewer, Tester).
- **Goal:** Solve hallucination and redundancy through layered tasks (Planning -> Classification -> Code Gen -> Execution -> Result).

### 2. AI NPC System
- **Tech Stack:** HTN (Hierarchical Task Network) + LLM (Large Language Model).
- **Capabilities:**
  - Real-time voice generation.
  - Dynamic dialogue and plot advancement.
  - Personalized behavior based on environment.
  - "Human-like mind simulation".

### 3. Full-Element Generation (UGC/AIGC)
- **Concept:** 0-code generation of game elements by players.
- **Scope:** NPCs (image-to-3D/character), Props (weapons/items), Plot, Logic/Gameplay.

## Flagship Product: Magiscapes (麦琪的花园)
